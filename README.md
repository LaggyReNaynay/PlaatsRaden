# PlaatsRaden

Niet gokken. Oplossen.

## Eerste versie
React + TypeScript + Vite + Leaflet.

## Lokaal starten
```bash
npm install
npm run dev
```

## Deploy op Vercel
- Upload deze bestanden naar GitHub
- Importeer de repository in Vercel
- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`
