# PlaatsRaden

Eerste professionele starter van het geografische puzzelspel.

## Lokaal draaien
1. Installeer Node.js
2. Run: `npm install`
3. Run: `npm run dev`

## Let op
Deze eerste versie gebruikt OpenStreetMap embed als tijdelijke kaartlaag. De uiteindelijke game moet satellietbeelden gebruiken via een geschikte kaartprovider.
